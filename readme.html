    4.报错“found character '\t' that cannot start any token”

    原因是在.yaml文件中不能使用tab键进行缩进。

    5.latex公式无法显示

    原因可能是在mkdocs中，<>中的反括号前需要加\，且每一个都要加。

    6.如何显示Tips、Note、Warning等块

    具体的规则是：!!!+space+Tip或Note或Warning。内容文字需要在下一行与Tip等缩进相同的位置开始书写。

    例如： 

    !!! Tip
    这是一个Tip。

    Tip

    这是一个Tip。

    9.解决Latex图片移动到下一页的问题

    参考：Latex控制图片位置

    使用float包并在图片语句后使用H选项

    \usepackage{float} 
% ... 
\begin{figure}[H] 
foo 
\end{figure}
